## Chrome

#### Shortcuts

| Devtools快捷键 | 用法 |
| ---  | --- |
| `F12`或`Ctrl-Shift-i`| 打开/关闭开发者工具 |
| `Ctrl-Shift-J` | 打开/关闭控制台 |
| `Ctrl-Shift-C` | 选择页面元素 |
| `Ctrl - F ` | 页面查找 |
| `Ctrl - R ` | 刷新页面 |
| `Ctrl - L` | 选中地址栏 |
| `H` | 隐藏文件 |
| `Ctrl-Space` | 自动补全 |	
| `Ctrl-M` | 光标转到匹配的括号 |
| `Ctrl-0 `| 恢复默认字体大小 |
| `Ctrl-Shift-M`| 切换到手机模式 |
| `Ctrl-Alt-在箭头上单击` | 展开/折叠节点及其子元素 |
| `ctrl-backspace` | 控制台向后删除一个单词 |
